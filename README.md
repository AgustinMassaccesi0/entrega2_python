Ejercicios Python

Resolución de ejercicios en Python realizados en notebooks de Jupyter con funciones reutilizables en el directorio src/.

Requisitos
Python 3.13.12
pip

Instalación
1. Clonar el repositorio
bash: git clone cd

2. Crear y activar el entorno virtual
Linux/macOS: bash: python3 -m venv venv source venv/bin/activate

Windows: bash: python -m venv venv venv\Scripts\activate

3. Instalar dependencias
bash: pip install -r requirements.txt

Estructura del proyecto 
## Estructura del proyecto

```text
entrega2_python/          # Repositorio remoto clonado
├── src/                  # Carpeta con las funciones utilizadas en los ejercicios
├── notebooks/            # Carpeta con los ejercicios realizados en Jupyter Notebook
├── requirements.txt      # Dependencias del proyecto
└── README.md             # Instrucciones de uso
```

Ejecución
Con el entorno virtual activado, ejecutar el siguiente comando para abrir Jupyter:

bash:  jupyter notebook notebooks/
