Ejercicios Python

Resolución de ejercicios en Python realizados en notebooks de Jupyter con funciones reutilizables en el directorio `src/`.


## Requisitos

- Python 3.13.12
- pip

-----------------------------------------------------------------------------------------------------------------------------

## Instalación

### 1. Clonar el repositorio

git clone <url-del-repositorio>
cd <nombre-del-proyecto>


### 2. Crear y activar el entorno virtual

**Linux/macOS:**
bash
python3 -m venv venv
source venv/bin/activate


**Windows:**
bash
python -m venv venv
venv\Scripts\activate


### 3. Instalar dependencias

bash
pip install -r requirements.txt

-----------------------------------------------------------------------------------------------------------------------------


## Estructura del proyecto


entrega2_python/
├── src/                        # Funciones y módulos reutilizables
│   └── ...
├── notebooks/                  # Ejercicios resueltos en Jupyter Notebook
│   └── ejercicio_n.ipynb       # n = (1-10)
├── requirements.txt
└── README.md


-----------------------------------------------------------------------------------------------------------------------------

## Ejecución

Con el entorno virtual activado, ejecutar el siguiente comando para abrir Jupyter:

bash
jupyter notebook notebooks/
